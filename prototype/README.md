# Agent Loop Runbook Builder

Source: [[Cursor Loops as Operational Agent Runbooks]]

## What it demonstrates
A static builder for flaky-test, post-deploy, and dependency-upgrade loops with copyable prompts and safety gates.

## How to run
Open `agent-loop-runbook-builder.html` in a browser. It is self-contained and stores the latest generated handoff in localStorage.
